# Evidence Handling

- This `evidence/` folder is a working area for local evidence and raw telemetry used during investigations.
- Do NOT commit raw, unsanitized telemetry to the public repository.
- Store sanitized test samples under `tests/sample-data/`.
- If you need to keep private evidence, move it outside the repository (for example, a local secure archive) and add any retained evidence files to `.gitignore`.
